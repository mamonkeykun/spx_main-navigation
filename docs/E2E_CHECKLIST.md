# E2E Checklist

## 基本レンダリング
- [ ] Communication Siteでナビが表示される
- [ ] Team Siteでナビが表示される
- [ ] Hub Siteでナビが表示される
- [ ] SharePoint標準ナビが非表示になる（hideSharePointNav=true時）
- [ ] ロゴ画像が正しく表示される
- [ ] ロゴなし設定でエラーが出ない

## ドロップダウン
- [ ] フォルダラベルにホバーするとドロップダウンが開く
- [ ] フォルダラベルをクリックするとドロップダウンが開く
- [ ] ESCキーでドロップダウンが閉じる
- [ ] ドロップダウン外クリックで閉じる
- [ ] 縦レイアウトで単列表示される
- [ ] 横レイアウトで3列グリッド表示される
- [ ] アイテムのdescriptionがサブテキストで表示される
- [ ] openInNewTab=trueのアイテムが新しいタブで開く

## ターゲティング
- [ ] allowedGroups=[]のアイテムが全ユーザーに表示される
- [ ] allowedGroups設定済みアイテムがグループ外ユーザーに非表示になる
- [ ] 非表示アイテムの位置にスペースが残らない
- [ ] フォルダが非表示の場合、その子アイテムも非表示になる
- [ ] ゲストユーザーにはallowedGroups=[]のアイテムのみ表示される

## パンくずリスト
- [ ] 現在ページのパスが正しく表示される
- [ ] 親セグメントがクリッカブルリンクになる
- [ ] トップページではHomeのみ表示される
- [ ] マッチするNavItemがない場合はHomeのみ表示される

## クロスサイトナビ
- [ ] sourceURLを設定すると別サイトのNavリストを読む
- [ ] sourceURLのサイトにアクセス権がない場合エラーメッセージが表示される
- [ ] タイムアウト時にローカルサイトのナビにフォールバックする
- [ ] 接続テストボタンが成功/失敗を正しく表示する

## Settings Web パーツ
- [ ] カラー変更が即時ナビに反映される（ページリロード不要）
- [ ] フォント変更が即時反映される
- [ ] 設定がページリロード後も保持される
- [ ] エラー時にエラーメッセージが表示される

## Nav Item Manager
- [ ] フォルダの追加・編集・削除ができる
- [ ] アイテムの追加・編集・削除ができる
- [ ] ドラッグ&ドロップでアイテムの順序が変わる
- [ ] 権限設定でグループを選択・保存できる
- [ ] 「全員に表示」トグルで権限が解除される

## モバイル
- [ ] 768px以下でハンバーガーメニューが表示される
- [ ] ドロワーが左からスライドインする
- [ ] アコーディオンでフォルダを展開できる
- [ ] バックドロップクリックでドロワーが閉じる
- [ ] ESCキーでドロワーが閉じる

## アクセシビリティ
- [ ] Tabキーで全ナビ要素にフォーカスできる
- [ ] Enterキーでドロップダウンが開閉する
- [ ] スクリーンリーダーがナビ構造を正しく読み上げる
- [ ] フォーカストラップがモバイルドロワーで動作する

## ビルド
- [ ] `gulp bundle --ship` がエラーなく完了する
- [ ] `gulp package-solution --ship` がエラーなく完了する
- [ ] .sppkg がApp Catalogにアップロードできる
- [ ] CI (GitHub Actions) がすべてのジョブを通過する
